# LifeSync – Student Life Management System

## 📖 Project Overview

LifeSync is a web-based Student Life Management System designed to help students organize and manage their daily academic and personal activities efficiently. The system provides a centralized platform for tracking attendance, managing tasks, monitoring expenses, and maintaining important student information.

This project was developed as part of the Summer Internship Project (SIP).

---

## 🎯 Objectives

* Simplify student life management.
* Improve productivity and organization.
* Provide an easy-to-use interface for managing daily activities.
* Integrate multiple student utilities into a single platform.

---

## ✨ Features

### Attendance Management

* Record and track attendance.
* View attendance details and updates.

### Task Management

* Add, update, and delete tasks.
* Track pending and completed tasks.

### Expense Tracker

* Record daily expenses.
* Monitor spending habits.

### Dashboard

* Display important information in one place.
* Provide quick access to all modules.

### User Management

* User registration and login functionality.
* Secure access to personal data.

3. Open the project folder.
4. Open `index.html` in any modern web browser.
5. Navigate through the modules using the provided interface.
